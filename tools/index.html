Linux Distros for Android Development

You will need a Linux distribution to compile android ROMs, whether it is Fortune OS, or any other ROM. Below you will find some Linux distributions that we recommend for compiling Android from source:
    BBQLinux           Ubuntu           Debian                LXLE         

Android SDK

If you chose BBQLinux, you will not need to install the Android SDK. If you are using another distribution, click the button below to learn how to install the SDK.
